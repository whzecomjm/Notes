- [Bug 反馈](http://hexo.io/docs/troubleshooting.html) 
- [GitHub 项目](https://github.com/hexojs/hexo).

## 常用指令

``` bash
$ hexo new "My New Post" //创建一个新文章
$ hexo server //调试
$ hexo generate //生成静态文件
$ hexo deploy //发布
$ hexo clean //清空缓存 (更换主题以后常用到)
```
